## [2.1.0] - 2026-02-01

### Added
- Improved support for HAML.
- Switched to `@herb-tools/tailwind-class-sorter` engine for better Tailwind 3/4 support.
- Modernized development environment (TypeScript 5, Jest 29).

### Changed
- Removed manual `headwind.defaultSortOrder` configuration (now automatic).
- Updated documentation and added a development guide.
